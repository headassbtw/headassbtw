Amplitude 2
This page is unlisted on purpose
Main Idea
A clone of Amplitude (2016, PS3/PS4) Designed to be PC-friendly, essentially just a close approximation of the game, but that runs natively.

The goal is to have a high-performance close approximation of Amp16's gameplay, for lower-end devices that can't run RPCS3.
(Also for me to have fun making)
"Engine""
Amplitude 2 uses a Revolutionary low-overhead engine designed from scratch to achieve maxumum something.
Available for licensing, better than Unreal 6.

Ok but actually, i fucking hate the idea of games needing to market themselves with an "engine" and I guess by now you can tell I'm not using a pre-made one.
I'm using OpenGL for graphics, SDL2 for input, and OpenAL for audio.
Platform support
Primarily PC (linux, windows, and OSX, in that order) but I'm going to try to release Android and Nintendo Switch builds.
Features
Features that stand out from other rhythm game clones, or the original game itself:
  • Developer Console
  • Shitass Graphics Mode
  • Keyboard Support
